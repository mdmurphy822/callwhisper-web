# Accessibility Statement

CallWhisper is committed to ensuring digital accessibility for all users. We have designed this application to meet **WCAG 2.2 Level AA** standards.

## Accessibility Features

### Keyboard Navigation
- **Skip to Content**: Press Tab on page load to reveal a "Skip to main content" link
- **Full Keyboard Access**: All features accessible without a mouse
- **Keyboard Shortcuts**: Press `?` to view available shortcuts
  - `Ctrl+Space` / `Cmd+Space`: Start or stop recording
  - `Ctrl+S`: Save transcription
  - `Ctrl+K`: Focus search
  - `Ctrl+R`: Refresh device list
  - `Ctrl+Shift+R`: Reset to idle
  - `Escape`: Close dialogs or cancel operations

### Screen Reader Support
- Tested with NVDA screen reader
- All status changes are announced automatically
- Form fields have descriptive labels and instructions
- Error messages are announced immediately

### Visual Accessibility
- **High Contrast**: Text meets WCAG contrast requirements
- **Focus Indicators**: Clear visible focus outline on all interactive elements
- **Reduced Motion**: Animations are disabled when you have "Reduce motion" enabled in your operating system

### Text and Layout
- Text can be resized up to 200% without loss of functionality
- Interface adapts to different screen sizes and orientations

## Known Limitations

- The application is currently available in English only
- The "Reset" button does not have a confirmation dialog
